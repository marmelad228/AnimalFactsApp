# Animal Facts App

Консольное REST API приложение на языке C# с интеграцией контроля версий Git.

## Реализованный функционал
* Асинхронные GET-запросы к внешним серверам через HttpClient.
* Обработка ошибок сетевого соединения с помощью блоков try-catch.
* Десериализация JSON-ответов библиотеки Newtonsoft.Json.
* Контроль версий проекта: использование веток (Git Flow) и информативных коммитов.
